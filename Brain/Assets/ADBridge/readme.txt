ADBridge

v1.0.0   2019-12-13   LingXin
Feature
    1. Mediation advertising platform:
        - IronSource
        - Mopub
HowToUse
    1. Creat empty gameobject at hierarchy root in launch scene, then add component "Loom" in this gameobject.
    2. Editor -> ProjectSettings -> Player -> OtherSettings -> ScriptingDefineSymbols
        - add "AD_IRONSOURCE", if you use IronSource
        - add "AD_MOPUB", if you use Mopub
